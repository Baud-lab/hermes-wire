In the `codes` folder you are going to find a main R script named `characterization_heritable.R`, which is the additional script containing the biggest part of the parallel analyses performed based on Step2 inputs and outputs.
Run this code using the command line:

```
Rscript characterization_heritable.R \
  --input_tsv filtered_VCs.tsv \
  --outpdf ../Results/characterization_heritable.pdf \
  --subset_id "Full cohort"
```
